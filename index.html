üì± Creating QR Codes

To create your physical scavenger hunt:

  1. Host these HTML files on a web server, Google Drive, or use a service like GitHub Pages (free)
  2. Use a free QR code generator like qr-code-generator.com
  3. Generate QR codes for each team's start page URL
  4. Print the QR codes and place them around your school
  5. Each correct answer will show a link to the next question - generate QR codes for those too!

Alternative: Share the links directly via text/email if you prefer a digital-only hunt
